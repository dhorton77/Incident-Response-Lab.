Incident Response Lab – Detection & Analysis (Windows)
Prerequisites

Before running the scripts in this project, please ensure you have the Sysinternals Suite downloaded and extracted. You can download it directly from Microsoft:
Download Sysinternals Suite

Overview

This lab demonstrates a structured incident response process on a Windows system. It includes memory acquisition, IOC identification, network analysis, and system log inspections using tools like Sysinternals and PowerShell.
